핵심 원칙 — 기획안을 그대로 조립한 로우파이

기획안의 화면 흐름(screens[])·저장 정보(data_model[])를 컴포넌트로 조립한 구조 확인 화면. 색·브랜딩은 입히지 않고(뉴트럴 그레이, shadcn 계열 컴포넌트 톤), CTA만 검정으로 위계 하나만 남긴다. 실제 데이터가 저장되는 지점("발주 데이터로 저장돼요")을 라벨로 표시해 기획-데이터 연결을 보여준다.

규칙
  • 화면 탭 = screens[] 순서 그대로 · 핵심 화면(is_core)부터 검토 유도
  • 텍스트는 실제 카피(기획안 값) — 로렘 입숨 금지. 이미지 자리만 IMG 점선 박스
  • 수정 = 채팅("장바구니를 항상 보여줘") — 구조 반영 후 이 화면 갱신
  • [좋아요, 이대로 만들기 시작] → 디자인 브리프의 토큰을 입혀 생성 진행
  • 구현 옵션: 자체 로우파이 컴포넌트 셋 또는 shadcn/ui 뉴트럴 테마 렌더 — 협의

진입: 기획안 완성 → [와이어프레임으로 먼저 볼래요] · 전체 명세 §2.5.
