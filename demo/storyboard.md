# AiApp 제품 영상 데모 기획 — "좀비웹 리뉴얼 (ETL)"

죽어있는 옛날 홈페이지를 AiApp이 데이터째로 가져와(ETL) 운영 가능한 서비스로 되살리는 스토리. 데모 소재는 B2B 제조업체 **(주)OO정밀** (CNC 정밀가공, Since 1987).

**핵심 메시지**: "주소만 알려주세요 — 데이터는 그대로, 디자인과 운영은 오늘에 맞게."

## Before 디자인 기획 (`legacy-b2b.html`)

**설계 원칙**: 조롱이 아니라 공감 — "우리 회사 홈페이지 같다"는 반응이 목표. 당대 기법(테이블 레이아웃, font 태그, 레이어 팝업)을 실제로 사용해 재현.

| 요소 | 의도 |
|---|---|
| 🧩 깨진 Flash 배너 (`main_banner_2008.swf`) | 좀비웹의 상징. 첫 3초 안에 시대를 규정 |
| "리뉴얼 준비중" 팝업 (2015.03) | 데모의 펀치라인 — 그 리뉴얼을 AiApp이 끝내줌 |
| 깜빡이는 "FAX 견적서 환영!!" | S8(견적 인박스)과 수미상관 |
| 견적 게시판 512건 · 비밀글 · 스팸 | "죽은 사이트에도 살아있는 데이터"의 증거 = ETL 명분 |
| 제품 품번(DP-####) · 설비현황 표 | After에서 그대로 이전되는 구조화 데이터 |
| 만료된 인증서(2006) · Last Update 2014 · IE 7 최적화 | 방치의 디테일 (공감 포인트) |
| 굴림 12px · 900px 테이블 · 네이비 그라디언트 GNB | 2000년대 중소기업 홈페이지 표준 문법 |

**After와의 연속성 장치**: 네이비 브랜드 컬러와 주황(견적 박스) 포인트, 제품 품번, "1987 창립"을 After가 그대로 계승 — "다른 회사가 된 게 아니라 되살아난 것"을 시각적으로 증명.

## 데모 러너 사용법 (`index.html`)

**완전 자동 시퀀스 (~53초)** — 브라우저로 열면 바로 재생 시작. F11(전체화면) 후 녹화만 하면 됨. 화면에 컨트롤 없음. `R` 키 = 처음부터 다시.

자동 진행 순서:

1. **Before 문제 강조** (0–14s) — 풀스크린 좀비웹 위에 스포트라이트+콜아웃 4연타: ❌ Flash 지원 종료 → ❌ 제품 사진 "이미지없음" → ❌ 리뉴얼 준비중(2015~) → ⚠ 그런데 견적은 지금도 온다
2. **링크 한 줄** (14–18s) — AiApp 컴포저가 떠오르고 URL 타이핑 → 전송
3. **스튜디오 진입** (18–25s) ⭐ — **풀스크린이던 옛 사이트가 AiApp 스튜디오의 미리보기 패널 안으로 도킹**. 왼쪽 채팅에 분석 로그가 차오름 (✓ 제품 4종 · 설비 11대 · 게시판 512건 · 연혁·인증 쫘라락) + 진행 콘솔 카드
4. **전환** (25–28s) — **미리보기 안에서** 주황 엣지 와이프로 After 리빌, 주소가 aiapp.site로, 배지 LIVE + 채팅에 완료 보고 카드
5. **After 둘러보기** (29–34s) — 미리보기 안 자동 스크롤: 제품 → 온라인 견적 폼
6. **관리** (34–44s) — **푸시 알림 카드**(주황 파동 링 + 회사·품목·첨부 정보)가 우상단에 등장, 데이터베이스 탭에 주황 "1" 배지(파동) + 자막 *"리뉴얼 첫날 — 벌써 새 견적이 도착했습니다"* → 약 2초 머문 뒤 **탭이 자동 전환**되며 견적 인박스: NEW 행 + 배너 + [회신] 펄스
7. **채팅 수정** (46–56s) — 6장 자막이 걷힌 뒤 한 템포(약 1.4초) 쉬고 → 자막 *"홈페이지 문구 하나 바꾸려면 — 개발사에 연락하고, 한참 기다리셨죠?"* → 미리보기 탭 복귀 → 채팅 "메인 문구를 '당일 견적 회신'으로 바꿔줘" (**말풍선 글로우 + 미리보기 딤으로 채팅 포커싱**) → **히어로 문구가 실시간으로 바뀌며 주황 하이라이트** + "이전 버전 저장 — 직전 상태로 되돌리기 가능" (되돌리기는 현재 1회 지원이라 '언제든' 표현 금지)
8. **제품 추가** (56–62s) — 채팅 "신제품 '유니버설 커플링 DP-7120'도 추가해줘" (**채팅 포커싱**) → 미리보기가 제품 섹션으로 스크롤, **신제품 카드가 NEW 배지 + 주황 링과 함께 그리드에 등장** (실사진 product-05.png — "보내주신 사진으로 카드까지") → 자막 *"신제품 등록까지 — 코드 없이, 기다림 없이"*
9. **공지 올리기** (62–70s) — 채팅 "8/4~8/8 여름휴가 공지 올려줘" (**채팅 포커싱**) → **사이트 중앙에 모던 공지 팝업이 스프링 등장** (여름휴가 휴무 안내 카드 — Before의 2015 레이어 팝업의 '오늘 버전'으로 수미상관) → 자막 *"마지막 공지가 2015년이었죠 — 이제 말 한마디면 됩니다"*
10. **엔드카드** (70s~) — "오래된 홈페이지, 오늘의 서비스로."
