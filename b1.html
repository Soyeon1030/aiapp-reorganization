핵심 원칙

입력은 아이디어 한 문장이 전부다. 프로젝트명·도메인·템플릿·레이아웃은 묻지 않는다 — AI가 아이디어에서 도출한다. 확인(설계안)은 옵션이 아니라 항상 거치는 기본 동작.

흐름
사용자 아이디어 입력 한 문장 + (선택) 사진·참고 링크
→
검증 최소 10자 미달 시 구체화 유도 마이크로카피
→
AI → 승인 대기 🔒 설계안 대화 blueprint — 이름·기획 확인
→
전이 스튜디오(B2) 생성 → 미리보기 → 출시
프로젝트명 규칙 — 사용자가 짓지 않는다

B1에 이름 입력이 없는 건 의도다. AI가 입력 내용을 파악해 주요 주제를 제목(서비스명)으로 자동 결정하고, 설계안 ① 기획안 카드에서 확인·수정할 수 있다. "동네 반려견 산책 예약…" → 멍멍 산책 예약.
